How to download and install Genit

First install Ruby and rubygem

As Genit is written in Ruby, you need the Ruby interpreter to run it. You need rubygem too. Rubygem is a marvelous Ruby application that acts as an installer.
Go to www.ruby-lang.org and follow the instructions...

Download and install Genit

Thanks to rubygem, downloading and installing Genit is as simple as opening a console and type:

  gem instal genit

All other dependencies will be downloaded and installed automatically.

Smoke test

This step is optional but performing the smoke test is a good thing to be sure the installation is well done.
Open a console window and type: 


  genit create test
  cd test
  genit compile

Then go to the test/www/ folder and double click the index.html file.
That's it !
